# Battlesnake utilities

A small set of utility classes for a python [battlesnake](https://docs.battlesnake.com/)

## Description

A set of classes for battlesnake representing the game state, the board, snakes, etc.

## Getting started

### Dependencies

* python 3.9
* pandas

### Installing

* git clone
* poetry install
* python -m pytest
